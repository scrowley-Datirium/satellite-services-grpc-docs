(define transitions out of, as into this state) */ ready: { /** define what event triggers what */ on: { /** if STARTED event received, transition to 'online' */ STARTED: { target: 'online' }, }, }, /** define what happens in 'online' state */ online: { /** when this state is entered, run 'sendReadyToChild' action. downloading

Default value : 'downloading'

downloading taskType indicates a sample needs to have it's inputs downloaded before having its analysis run
finishedDownloading
Type : string
Default value : 'finished_downloading'

finishedDownloading taskType indicates that a download has finished. transitions into 'processing'
newSample
Type : string
Default value : 'new_sample'

TaskTypes are what kind of tasks is given to the satellite from task-queue-machine Tasks are related to sample downloads/analysis runs
processing
Type : string
Default value : 'processing'

processing taskType indicates that an anlysis is running (using cwltool, airflow, ...)
rerunSample
Type : string
Default value : 'rerun_sample'

rerunSample taskType indicates that a sample's analysis is to be rerun (using cwltool, airflow, ...). Transition to 'processing'
updatedSample
Type : string
Default value : 'updated_sample'

updatedSample taskType indicates that a sample's inputs/preview/ something has been updated. checks if downloading is necessary before checking if rerunning is necessary
